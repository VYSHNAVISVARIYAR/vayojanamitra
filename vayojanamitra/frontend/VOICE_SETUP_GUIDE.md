# Voice Features Setup Guide

## 🎤 How to Enable Voice Features in Your Browser

### **For Chrome/Edge Users:**

1. **Enable Microphone Access:**
   - Click the **🎤 microphone icon** in the chatbot
   - Click **"Allow"** when prompted for microphone permission
   - Make sure the site is not in incognito mode

2. **Enable Speech Synthesis:**
   - Go to **Browser Settings** → **Privacy and security** → **Site Settings**
   - Find **vayojanamitra** (localhost:5173) in the site list
   - Set **Microphone** and **Sound** permissions to **"Allow"**
   - Refresh the page and try again

### **For Firefox Users:**

1. **Microphone Permission:**
   - Click the **🎤 microphone icon** in the chatbot
   - Click **"Allow"** in the permission prompt
   - Check that the site is not in Private Browsing mode

2. **Speech Synthesis:**
   - Go to **Options** → **Privacy & Security** → **Settings**
   - Scroll to **Permissions** section
   - Set **"Use audio services in all sites"** to **"Allow"**
   - Refresh the page

### **Troubleshooting:**

**If voice still doesn't work:**
- Try using a different browser (Chrome/Edge work best)
- Clear browser cache and cookies
- Disable browser extensions that might block audio
- Make sure you're not in incognito/private mode
- Update your browser to the latest version

### **What Each Voice Feature Does:**

**🎤 Voice Input:** 
- Lets you speak instead of typing
- Converts your voice to text messages
- Requires microphone permission

**🔊 Voice Output:**
- Lets you hear the chatbot's responses
- Converts text to spoken words
- Requires speech synthesis permission

### **Still Having Issues?**

Voice features are **optional** - you can still use the chatbot by typing messages even if voice doesn't work!
